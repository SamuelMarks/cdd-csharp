# OpenAPI 3.2.0 Compliance

This project strives for complete compliance with [OpenAPI 3.2.0](https://raw.githubusercontent.com/OAI/OpenAPI-Specification/refs/heads/main/versions/3.2.0.md).

Currently supported features:
- Info, Server, Components objects
- PathItem, Operation, Parameter
- RequestBody, Responses
- Schema (with Discriminator, Reference)
- Example, ExternalDocs
- Tag, OAuthFlows, SecurityScheme

A complete gap analysis will be updated continuously.
